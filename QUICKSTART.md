# ⚡ شروع فوری - Quick Start

## 🎯 فقط 3 دقیقه!

### مرحله 1: اجرا کنید
```bash
# Windows
start.bat

# Mac/Linux
bash start.sh
```

### مرحله 2: تنظیم کنید
- `.env.local` فایل را ویرایش کنید
- توکن‌های خود را اضافه کنید

### مرحله 3: ببینید!
- مرورگر: http://localhost:5000
- ثبت‌نام کنید و شروع کنید
